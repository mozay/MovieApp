# MovieApp

The MovieDb API

<b>Tech Stack</b>
* Kotlin %100
* MVVM Clean Architecture
* Coroutines
* Jetpack (LiveData, ViewModel, DataBinding, Navigation)
* Retrofit
* Dagger Hilt
* Kotlin DSL (for Gradle)
* Infinitely scroll
